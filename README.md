## 📈 Daily Progress Log

### 🗓️ June 28, 2026
* **Course:** Introduction to Sustainable Finance Taxonomies
* **Module:** Module 1 — An Overview of Sustainable Finance Taxonomies

#### 🔍 Core Concepts Learned:
* **Definition of Taxonomy:** A classification system that defines which economic activities are environmentally sustainable. It provides a common language for investors, companies, and policymakers.
* **The "Why":** It prevents **greenwashing** (companies falsely claiming to be eco-friendly) and channels private capital into truly sustainable projects.
* **Key Global Drivers:** Driven by the Paris Agreement (limiting global warming to 1.5°C) and the UN Sustainable Development Goals (SDGs).
* **Taxonomy Types:**
  - *White/Green List:* Defines clearly sustainable activities.
  - *Transition Taxonomy:* Helps high-carbon industries move toward greener operations.

#### 💡 Key Takeaway:
* Taxonomies convert environmental goals into **standardized data metrics**. Financial data analysts use these metrics to screen investment portfolios and calculate corporate sustainability scores.

### 🗓️ June 29, 2026
* **Module:** Module 2: Taxonomies in Practice: The EU and China

#### 🔍 Core Concepts Learned:
* **The EU Framework:** Driven by investor disclosure rules using strict, science-based Technical Screening Criteria (TSC) and mandatory "Do No Significant Harm" (DNSH) safeguard testing.
* **The China Framework:** Driven by state industrial policy using the Green Bond Endorsed Project Catalogue, which operates as a direct "Whitelist" for instant project eligibility.
* **Policy Channels:** The EU embeds its taxonomy into corporate transparency laws, while China links its framework to central bank incentives and commercial bank credit tools.
* **Interoperability Bridge:** The Common Ground Taxonomy (CGT) acts as a technical mapping tool to find overlapping green criteria between both regions for cross-border investments.

#### 💡 Key Takeaway:
While the EU relies on a threshold-based disclosure system to enforce pure-green market transparency, China uses a project-based whitelist system to accelerate state-led transition finance and direct industrial upgrades.
